# Medico – Chronic Health Companion App  
**SOEN 357 – UI/UX Design Case Study**

## Overview
This repository contains the UX/UI case study for **Medico**, a conceptual health companion mobile application designed to support individuals managing chronic health conditions. The project was completed as part of the SOEN 357 course at Concordia University.

The goal of this project was not to build a functional application, but to demonstrate the full UX design process, including research, ideation, wireframing, prototyping, and usability testing.

---

## Live Case Study
The case study is hosted using **GitHub Pages** and can be accessed at:

**https://yuioytffhio.github.io/medico-ux-case-study/**

---

## Project Scope
This project focuses on:
- User research and survey design
- Persona creation
- User journey mapping
- Sketching and wireframing
- Visual design decisions
- Interactive prototyping using Figma
- Usability testing planning
- Reflection and iteration

No backend, frontend logic, or real data storage is implemented. All designs are conceptual and intended for academic purposes only.
